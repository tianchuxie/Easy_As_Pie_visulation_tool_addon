Installation

Please note that this product has not yet been published and is still undergoing the testing stage. As such, the EasyAsPie Mozilla plugin is currently only available as a downloaded Mozilla/Firefox Browser Extension Archive file through the CMSC435 Wildcard group repository. This will require the login and credentials of one of the project CMSC435 Wildcard group members.

 

Follow the steps below to complete the installation and setup process.

 

  1. Download the add-on package.
  2. If Mozilla Firefox is not already downloaded on your system, follow the installation and setup steps by selecting this link. To update your Mozilla Firefox version, follow the instructions at the following link.
  3. If Java is not already downloaded on your system, follow the installation and setup steps by selecting this link.
  4. Navigate to “about:addons” in a Mozilla Firefox tab.  
  5. Select and drag your downloaded “easyaspie-1.1-an+fx.xpi” file onto the loaded Mozilla Firefox page from step 6.
  6. Click on the “Install” button (outlined in green below) in the popup window on the upper-left hand corner.  
  7. Upon the completion of a successful installation, the popup image shown in the figure to the right should appear.
  8. Verify that the extension button, , appears in the extension toolbar.

 

 Upon the completion of the procedure above,  EasyAsPie has been successfully installed and configured.

Demonstration as a web user:

 

We use a sunburst visualization model, which allows users to interact with a pie chart, rather than other commonly used forms, such as a scroll bar. The circular shape of the pie chart lets users locate their intended data efficiently while requiring minimal mouse movements. Our product displays all layers and components, unlike other products who focus on a single selected layer, increasing the cost of user readability. Another advantage users will acquire is that our product works by selectively clicking to expand and display concentric arcs requested by the user. This feature minimizes spatial volume and increases textual label readability.

Developer's Help Page (Server Side)

 

For the experienced developer: 

Because our target customer is the developer, we can assume the developer should have some basic knowledge of computer science technology. In other words, the developer should know well about the setting up their own host, loading the database and connecting the correct formatted JSON file in our product. 

We have provided the sample PHP file to help process the database: SamplePartition.php . This file reads the database in the host and partition into 3 arbitrary layers (it's object oriented, you can determine which should be partition). 

 

In particular, the developer can change their partition rule in this line of code.

Furthermore, the developer can also change the individual element on the final layer by changing different column names

 

We have also provided the demo page associated with the database:demopage.html . Our product should run only by visiting this web page, because we provide the connection to formatted JSON file. 

 

 

For the amateur developer:

Because EasyAsPie is temporarily running on UMD's VM (Vagabond),  it's better to run on the developer's own machine. So we have provided you an alternative solution to help you set up your own host and creating the customized database and partition rules: XAMPP. If the developer has proper CSV file (which stores the database), we suggested the developer to use PHPMyAdmin (an application in the XAMPP localhost) to direct load the CSV file to the local host by clicking "import". 

(This is just one of the easiest way to set up the host)
