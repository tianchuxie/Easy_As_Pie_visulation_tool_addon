EasyAsPie is an application geared towards medium-level website developers who desire to provide their customers with a better data visualization experience.  It is designed as an interactive pie chart that allows client side users to navigate through large organized data sets with minimal mouse actions.  And on the server side, it allows for seamless importing of databases for flexible visualization.  

 

Features include:

    • Stunning sunburst-style visualization of data sets
    • Interactive, allowing users to sort through the data as they please
    • Support for importing a mySQL database to visualize
    • Customizable organization patterns for the pie chart
    • Developed using SQL and Javascript, as well as HTML and CSS.
    • Compatible with Mozilla Firefox; the only thing needed to run is our add-on and Javascript!
